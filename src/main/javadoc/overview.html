Mockobor

Mocked Observable Observation - library to simplify some aspects of unit testing with java.

Mockobor can help by follow problems:

  • simulate sending of events (via java listeners) from mocked collaborator to tested object
  • check completely deregistration of listeners registered by mocked collaborator
  • (not implemented jet) collect and verify events sent from test object
  • (not implemented jet) synchrone start of `Feature`, `SwingWorker` etc. to simpler verify of result of (in production asynchrone) invocations

Javadoc entry point: {@link org.mockobor.Mockobor}
